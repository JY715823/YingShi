# Current Task - Stage 1 Design System and App Shell

## Goal
Establish the design system baseline and strengthen the global app shell for Yingshi.

## Scope
This stage only covers:
- theme tokens
- bottom navigation styling refinement
- photo module top navigation refinement
- photo module single-row top structure refinement
- clearer shell/page scaffolds
- preview-friendly UI organization

## Product intent
- The app should feel calm, soft, restrained, and durable.
- The current focus is not business completeness, but correct page identity and visual hierarchy.
- Photo page should feel light.
- Album page should feel softer and more content-oriented.
- Viewer and system media tool area are not implemented yet, but current shell should leave room for them.

## Must follow
- 底部导航：主页 / 照片 / 生活
- 照片模块顶部导航：照片 / 相册 / 回收站
- 照片模块顶部为单行结构：
  - 左侧：照片 / 相册 / 回收站
  - 右侧：系统媒体按钮 + 通知铃铛
- 页面气质优先，不要堆业务逻辑
- 不要提前接入后端、数据库、评论、真实 Viewer、真实系统媒体

## Done when
- The app builds and runs
- Theme tokens are better organized
- Shell and navigation structure are clearer
- The photo module shows the intended single-row top structure
- UI is still simple, but noticeably more aligned with the design draft
