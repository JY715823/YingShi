# Yingshi Android

Android client for Yingshi.

## Current status
Early bootstrap stage.
Focus:
- project structure
- docs
- navigation shell
- placeholder UI

## Tech stack
- Kotlin
- Jetpack Compose
- Material 3

## How to run
1. Open this project in Android Studio
2. Wait for Gradle sync
3. Run the `app` configuration on an emulator or device
